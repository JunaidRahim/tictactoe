## How to run ?
* **C:**  <br> `gcc tictactoe.c -o tictactoe` <br> `./tictactoe`

* **C++ :** <br> `g++ tictactoe.c -o tictactoe` <br> `./tictactoe`

* **Golang :** <br> `go run tictactoe.go`

* **Java :** <br> `javac tictactoe.java` <br> `java tictactoe`

* **Python :** <br> `python3 tictactoe.py`
